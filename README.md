# 像素众创

_xiàngsù zhòngchuàng_

Xiangsu Zhongchuang (XSZC) is a Collective pixel artwork.

## Install

```bash
pnpm i
```

## Run

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
